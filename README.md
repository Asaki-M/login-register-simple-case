# login-register-vue-node

## 实现功能

实现登录和注册两个功能，实现全栈，数据记录在数据库，前后端分离

------

## 项目部署

### 数据库

数据库表中类型都是为 varchar，建了简单表， user表结构如下：

username|password|email
---|:--:|---:
user|user|user@163.com
admin|admin|admin@163.com

### 项目初始化
```
npm install
```

### 后端部署和运行
```
cd service
npm install
node app.js
```

### 运行前端项目
```
npm run serve
```
